# 我的笔记 - 轻量记事本应用
一个简洁、美观、稳定的本地笔记工具，无需服务器，打开即用，数据永久保存在本地。

## 🌟 功能介绍
- 笔记新增、修改、删除
- 搜索标题/内容快速查找
- 9 种漂亮皮肤自由切换
- 记录创建时间与修改时间
- 按季节自动显示天气（稳定不报错）
- 数据本地存储，刷新不丢失
- 移动端/电脑端都适配

## 📦 部署方式
本项目可直接部署在 GitHub Pages 上，只需：
1. 将 index.html 放在仓库根目录
2. 开启 GitHub Pages
3. 访问即可使用

## 🔧 版本信息
- 当前版本：v1.0.2
- 更新内容：修复天气获取失败问题，优化稳定性
- 运行环境：纯 HTML + CSS + JS，无需依赖

## 📄 使用说明
1. 输入标题和内容，点击「保存」记录笔记
2. 点击「修改」可编辑笔记并记录更新时间
3. 顶部搜索框可快速筛选笔记
4. 点击彩色圆点切换背景皮肤
5. 数据保存在浏览器本地，卸载浏览器会清空
